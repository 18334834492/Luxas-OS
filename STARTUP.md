# 系统启动指南

## 快速开始

### 1. 编译系统
```bash
./build.sh
```

### 2. 运行系统
```bash
./start.sh
```

### 3. 清理编译产物
```bash
./clean.sh
```

## 系统架构

```
┌─────────────────────────────────────────┐
│           Kernel Lite (微内核)          │
│  - IPC、线程调度、内存管理、中断转发     │
│  - 目标: < 100KB                         │
└────────────┬────────────────────────────┘
             │ 共享内存
             ↓
┌─────────────────────────────────────────┐
│         Luxas Shell (特权桌面服务)        │
│  - 合成器、窗口管理、AI集成              │
│  - 目标: 端到端延迟 < 1ms                │
└────┬───────────────────┬─────────────────┘
     │                   │
     ↓                   ↓
┌──────────┐      ┌──────────────┐
│ Windows  │      │   Windows    │
│ 子系统   │←──→  │ DirectX转Vulkan│
│ (25个    │      │ 转译层        │
│ 系统调用)│      └──────────────┘
└──────────┘

┌─────────────────────────────────────────┐
│         本地 AI 集成模块                 │
│  - 量化模型加载                          │
│  - 对话、文件搜索、自动化                │
└─────────────────────────────────────────┘
```

## 配置说明

### 系统配置文件
`config/system_config.yaml`

主要配置项：
- 内核配置（镜像大小、功能开关）
- Luxas Shell配置（延迟目标、功能开关）
- Windows子系统配置（系统调用数量、特性）
- AI模块配置（模型格式、量化位数）
- 性能目标（内存占用、启动时间）

### 性能目标

| 指标 | 目标值 |
|------|--------|
| 开机内存占用 (完整桌面) | ≈ 280 MB |
| 冷启动到桌面 | ≤ 3 秒 |
| 完整系统体积 | 小于主流 Linux 发行版 |

## 调试

### 查看系统状态
```bash
# 查看进程列表
ps -e

# 查看内存使用
free -h

# 查看IPC状态
ipc_status
```

### 日志查看
系统日志输出到控制台，包括：
- 内核初始化信息
- 系统调用记录
- 组件加载状态
- 错误和警告信息

## 故障排除

### 编译错误
- 检查GCC版本（需要支持C99）
- 确保所有依赖已安装

### 运行时错误
- 检查配置文件是否正确
- 确保共享内存目录可写
- 查看错误日志
